# 校企实习岗位撮合平台

## 功能说明

- 学生投递岗位（学院未审核简历不能投递）
- 企业导师发布岗位、审核投递、录用学生
- 学院老师审核学生简历
- 二次确认：企业录用后，学生和企业可进行最终确认
- 时间线：投递详情页按时间线展示所有状态变更和二次确认记录

## 启动命令

```bash
cd internship-match

# 安装依赖
npm install

# 初始化数据库（种子数据）
npm run seed

# 启动服务
npm start
```

服务启动后访问：http://localhost:3000

## 失败用例说明

**【失败用例】未被录用的学生不能进行二次确认**

测试路径：`test/smoke.test.js` 第 213 行

场景：
1. 企业导师发布一个新岗位
2. 学生投递该岗位（状态为 pending，等待审核）
3. 学生在未被企业录用的情况下，直接尝试调用二次确认接口
4. 预期返回 400 错误，提示"只有已录用或企业已确认的投递才能进行学生确认"

验证点：
- HTTP 状态码：400
- 错误信息包含"只有已录用"关键字
- 确保状态机严格，防止越权操作
